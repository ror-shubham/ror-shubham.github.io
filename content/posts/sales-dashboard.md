OrangeSomething shut down its operations soon after Covid lockdowns. My journey with 6sense started soon thereafter.

My first project was to create a new app named Sales Dashboards from Scratch. I led the frontend development of this app.
- I was sole Frontend developer responsible for creating the Frontend architecture, setting up boilerplate code,
setting up the CI/CD pipeline, and creating the initial set of reusable components. Two other developers joined later to help with the development.
- The architecture we chose was `Javascript` + `React` + `GraphQL` + `Apollo`. The backend was built with Django
- Developing this app was a great experience where we spent 3 months in hackathon mode and were pushing patches till the last minute.
- The app went live in November 2020, and I was responsible for the maintaining and supporting the Frontend for the next 4.5 years till it was shut down in March 2025. RIP 🪦
- The app was a starting point for the sales team to get insights and recommendations for their accounts. It was a single page app with multiple tabs.
