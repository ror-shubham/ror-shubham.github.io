OrangeSomething shut down its operations soon after Covid lockdowns. My journey with 6sense started soon thereafter. My first project was to create a new app named Sales Dashboards from Scratch. I led the frontend development of this app. I was sole Frontend developer responsible for creating the Frontend architecture, setting up boilerplate code, setting up the CI/CD pipeline, and creating the initial set of reusable components.
